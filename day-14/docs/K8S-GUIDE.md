# DAY 14: Kubernetes Basics

## What is Kubernetes?

- Container orchestration platform
- Manages containerized applications
- Handles deployment, scaling, networking
- Declarative configuration (YAML)

## Key Concepts

### Pod
- Smallest deployable unit
- Contains one or more containers
- Shared networking (localhost)

### Deployment
- Manages pod replicas
- Handles rolling updates
- Ensures desired state

### Service
- Network abstraction
- DNS resolution
- Load balancing

### Ingress
- HTTP/HTTPS routing
- URL paths to services
- TLS termination
